<h1>Spoof Tool</h1>

An app to create web proxies using cloud platforms like Google, AWS, and more.

<br>

<p>To setup locally:</p>
<ul>
  <li>Clone repo into empty directory</li>
  <li>Run `npm i`</li>
  <li>Run `npm run start`</li>
</ul>
